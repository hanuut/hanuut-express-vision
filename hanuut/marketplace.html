Hanuut C2C Marketplace ("Hwanet")

Project & Testing Report

Project Goal & Achievement

The primary objective was to solve the "chicken-and-egg" problem of the empty marketplace by building an engaging, user-friendly C2C platform to drive initial user acquisition and content generation.

  • Re-architected the User Experience: Shifted from a confusing, tab-based system to a modern, industry-standard Bottom Navigation Bar, clearly separating B2C "Shops" from the C2C "Market".
  • Designed an Engaging UI: Created a full suite of minimalist and intuitive screens, including a social-media-style discovery feed, a guided ad creation flow, and a consolidated user hub ("My Hanuut").
  • Implemented a Robust Backend: Built a secure and scalable API with dedicated modules for `market` (ads) and `messaging`, decisively replacing previous insecure workarounds.
  • Developed a Feature-Complete Flutter Module: Implemented the new UI in Flutter using a clean, feature-driven architecture, powered by Riverpod for state management.
  • Added Personalization: Integrated a local user activity tracking system to personalize the UI, such as reordering the category rail based on user preferences.

What's Next? (Roadmap)

Tier 1: Polish & Core Experience Immediate

  • Real Category/Location Selection: Implement the dedicated selection screens for the ad creation and filter flows.
  • "Saved Ads" Functionality: Implement local database logic (Hive) to save and retrieve bookmarked ad IDs.
  • User Profile/Avatars: Allow users to upload a profile picture and view a public profile of other sellers.
  • Ad Management: Build the UI in "My Ads" to allow users to "Mark as Sold," "Delete," or "Edit" their posts.

Tier 2: Engagement & Growth Features Mid-Term

  • Real-time Messaging: Integrate WebSockets to provide instant message delivery, typing indicators, and read receipts.
  • Advanced Search & Filtering: Enhance backend and UI to support filtering by category, price, and condition.
  • Rich Share Previews: Use Open Graph meta tags to create clickable previews when deep links are shared on social media.

Tier 3: Monetization & Expansion Long-Term

  • Featured Ads ("Bumps"): Introduce an in-app purchase to allow users to bump their ad to the top of the feed.
  • Crossover Promotion: Create a pathway for successful C2C sellers to easily register as a professional "Shop".
  • Delivery Integration: Explore integrating "Tawsila" to offer on-demand delivery for C2C transactions.
